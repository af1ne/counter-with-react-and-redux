# Compteur numérique avec react et redux

Exercice de formation de la Wild Code School pour comprendre react et redux.

# Consignes

Créer un compteur React Redux

Dans une nouvelle application React, tu dois réutiliser le reducer de la quête précédente (Compteur en js natif).

Crée un composant CounterContainer qui affichera la valeur du compteur et qui comportera les différents boutons permettant de dispatch les différentes actions.

Après avoir dispatch toutes les actions dans l'application grâce aux différents boutons, utilise la fonction d'export dans Redux DevTools pour récupérer le fichier state.json. Ajoute ce fichier à la racine de ton projet.

Ajoute l'application dans ton github et poste le lien pour valider la quête.

# Critères de validation

    Le fichier counterReducer.js contient le code du reducer
    
    Le fichier CounterContainer.js contient le code d'affichage de la valeur du compteur et les boutons pour dispatch les actions
    
    Le fichier index.js comporte bien le code de création du store et le fournit bien à App via un Provider
    
    Le fichier state.json contient bien le nom de toutes les actions
